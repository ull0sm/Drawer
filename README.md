# 🥋 Drawer System

[![Python](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)
[![Docker](https://img.shields.io/badge/docker-supported-blue.svg)](https://www.docker.com/)

> **A comprehensive Python-based tournament draw system for karate events**

The Drawer System automates the creation of fair and unbiased single-elimination tournament brackets for karate competitions. It streamlines the entire process from data filtering to bracket generation, ensuring professional-quality tournament organization.

## 🚀 Getting Started

Quickly set up and run the system:
1.  **Clone:** `git clone https://github.com/ull0sm/Drawer.git`
2.  **Install:** `pip install -r requirements.txt`
3.  **Run:** `python automation/auto_run.py`

For detailed instructions, prerequisites, and troubleshooting, see the [**Quick Start Guide**](QUICKSTART.md).

## ✨ Features

- 🔍 **Intelligent Data Filtering:** Categorize players by gender, day, belt, and age.
- 👥 **Smart Group Creation:** Balanced groups by weight and school distribution.
- 📄 **Professional Bracket Generation:** Customizable PDF templates with randomized placement.
- ⚡ **Complete Automation:** One-click execution through `auto_run.py`.
